NopCommerce API

This project is built on Nopcommerce Opensource E-commerce database model It is meant to work on any Nopcommerce store without the need to keep updating the dependencies/methods with each new Nopcommerce version releases. The API is a standalone project and should be hosted on its own IIS instance pool--do not upload it to your store like regular plugin

NopCommerce API Plugin
